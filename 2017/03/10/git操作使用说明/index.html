git操作使用说明

发表于 2017-03-10 | 分类于 skill
express the semantic of the academic text
  • 第一次建库：在目标目录下打开git命令行，通过git clone “仓库地址”，在该目录下创建了一个与库名字

相同的文件夹，以后对该库中文件的更改和文件存储都在此文件夹中。

  • 对文件所有的操作都需进入该文件夹中，git命令行的操作指令参照linux系统的命令行，如ls、cd等。

在本地对文件夹内的内容做了改动后，通过git add “更改的文件”命令添加好更改的内容，没有提示warning

即表示成功。

  • git commit是对本次的修改做出说明解释，此时进入了vim编辑界面，s是插入，编辑好后，：wq是保存并退出

d是删除，通常用来删除产生的中间文件。

  • git push把本地文件push到远程github上。

  • 如果在github上做出修改，要与本地文件进行同步，则用git pull完成。
