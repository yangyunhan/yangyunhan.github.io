在一开始做这个效果的时候，一开始在rectangle外面又加了一层div，将div的position设置为relative（或者给body元素设置positive），发现垂直距离上的top和margin-top不起作用。
这个任务是在上述场景的基础上，在矩形左上右下角添加四分之一圆。两个圆的定位方法不同，圆一直接利用了margin的负值，圆二则无法使用margin负值，也无法使用float：right。
所以现在的效果不能根据矩形大小变换而始终定位到右下角。
