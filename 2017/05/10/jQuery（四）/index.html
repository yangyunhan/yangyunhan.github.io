jQuery（四）

Created at 2017-05-10 Updated at 2017-05-15 Category skill Tag notebook

1、伪类“:hover”用于当光标悬停在元素上时，改变元素的外形，在IE6浏览器中，仅用于超链接元素。对于其他元素可用jQuery的hover()方法。
hover()模拟光标悬停事件：光标移动到元素上触发函数1，移出元素触发函数2；它替代jQuery中的bind(“mouseenter”)和bind(“mouseleave”)，而不是替代bind(“mouseover”)和bing(“mouseout”)。因此当需要触发hover()方法的第2个函数时，需要用trigger(“mouseleave”)来触发，而不是trigger(“mouseout”)。
2、toggle()模拟鼠标连续单击事件[$().toggle(fn1,fn2…)]、切换元素的可见状态[$().toggle()]。
3、由于IE-DOM和标准DOM实现事件对象的方法各不相同，导致在不同浏览器中获取事件对象变得比较困难。jQuery进行了必要的扩展和封装，使得在任何浏览器中都能很轻松的获取事件对象以及事件对象的一些属性。即为函数添加一个参数$("element").bind("click",function(event){});当单击元素element时，事件对象就被创建了，处理函数执行完毕就被销毁。
4、stopPropagation()停止事件冒泡；preventDefault()阻止元素的默认行为。javascript中的这两个方法在IE浏览器中失效。jQuery对其进行了包装，可兼容各种浏览器，表示为event.stopPropagation()、event.preventDefault().return false是对同时调用两个方法的一种简写方式。
5、事件捕获与事件冒泡过程相反，并非所有浏览器都支持，jQuery不支持事件捕获，如要捕获需使用原生的javascript。
6、event.relatedTarget?
7、模拟操作trigger()方法，先给元素绑定事件，再用trigger函数触发事件。只触发绑定的事件，而不执行浏览器默认的操作，可以使用$(element).triggerHandle(事件)。可以添加事件命名空间，操作属于同一命名空间的事件对象，$().trigger(“click!”)的意思是匹配所有不包含在命名空间中的click方法。
8、用jQuery做动画效果要求在标准模式下，否则会引起动画抖动。DTD:<!DOCTYPE html PUBLIC “-//W3C//DTD XHTML 1.0 Transitional//EN” “http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
9、animate()是自定义动画函数，为了影响元素的top、left、bottom、right等属性，应先把元素的position样式设置为relative或者absolute。当animate方法中应用多个属性时，动画是同时发生的。
10、行距可以和字体、字体大小一起设置，font:12px/1.5em simsun,sans-serif，其中12px是字体大小，1.5em是行距，这种简写的方式必须是三个属性一起设置才行。
11、!import在CSS中表示优先级最高
12、text-intent:-9999px向左移除文字，其他元素将占据这个位置。
