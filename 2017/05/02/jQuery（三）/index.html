jQuery（三）

Created at 2017-05-02 Updated at 2017-05-10 Category skill Tag notebook

1、DOM操作分为3个操作：DOM Core（核心）、HTML-DOM和CSS-DOM。DOM Core并不属于JavaScript，任何一种支持DOM的程序设计语言都可以使用它。它的作用并非仅限于处理网页，也可以用来处理任何一种使用标记语言编写出来的文档，例如XML，Javascript中的getElementById()等方法都是Core的组成部分；HTML-DOM提供了一些更简明的记号来描述各种HTML元素的属性，例如document.forms、element.src等；CSS-DOM是针对CSS的操作。在JavaScript中，CSS-DOM的主要作用是获取和设置style对象的各种属性。
2、删除节点：

  • remove():当某个节点被删除后，该节点所包含的所有后代节点将同时被删除（包括这个元素绑定的事件），这个方法的返回值是一个指向已被删除的节点的引用，因此可以在以后再使用这些元素。

remove()方法也可以通过传递参数来选择性地删除元素。

  • detach():不会把匹配的元素从jQuery对象中删除，可以在将来再使用这些匹配的元素。所有绑定的事件、附加的数据等都会保留下来。
  • empty():清空节点，并清空元素中的所有后代节点。
    3、替换节点：
    replaceWith()和replaceAll()，如果是文本节点，可直接用text()。并且，如果在替换之前，已经为元素绑定事件，替换后原先绑定的事件将会与被替换的元素一起消失，需要在新元素上重新绑定事件。
    4、jQuery1.6中新增了prop()（处理固有属性？）和removeProp()，分别用来获取在匹配的元素集中的第一个元素的属性值和为匹配的元素删除设置的属性。
    5、JavaScript中的innerText属性并不能在Firefox浏览器下运行，而jQuery的text()方法支持所有的浏览器；text()方法对HTML文档和XML文档都有效。
    6、val():两个作用
  • 一是设置和获取元素的值，可用于文本框的提示效果（应用了focus()方法和blur()方法）；
  • 二是使select、checkbox、radio相应的选项被选中。示例：$("标签或选择器").val(["选项","选项"])只选一个也要加方括号。多次对同一个元素设置不同的默认值，最后前面设置的生效。attr()方法也可也实现一样的效果，如$("#single option:eq(1)").attr("selected",true)或者$("input[value=radio2]").attr("checked",true)或者$("[value=radio2]:radio").attr("checked",true)这个选择器中冒号前后可以对调。
    7、表单元素中单选按钮需设置属性name为同一个值，单选效果才起作用。
    8、表单元素select下拉菜单中，属性multiple的意思是允许一次多选。
    9、jQuery的bind()方法：
    向匹配元素添加一个或多个事件处理器，传入参数有事件、数据、执行函数。多个事件可以用空格分隔（多个事件调用同一个函数），也可以用大括号相隔（每个事件调用一个函数），一个大括号一个事件：函数。
    该函数的缺点是未来添加进来的元素不会携带已绑定的事件。
